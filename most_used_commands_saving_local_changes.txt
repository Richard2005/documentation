Git most used commands to save local changes

git status
to see if there are local changes 

git add -A|[fileName]
to add all the files/especific file, they are to push local

git commit -m 'message to show what's done'

push local the change local with a message to identify it.
PUSH it to the remote repository. 

List the branchs, there are enable
git branch 

Go to a branch
git checkout [branchName]

List all branchs there are available
git branch -l
To see the remote branchs, we need first made a checkout from the branch.
